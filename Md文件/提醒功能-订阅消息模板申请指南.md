# 微信小程序订阅消息模板申请指南

## 一、申请入口

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入您的小程序后台
3. 左侧菜单：**功能** → **订阅消息**
4. 点击 **"选用"** 或 **"申请模板"**

---

## 二、选择类目（如未设置）

订阅消息需要小程序有合适的服务类目才能申请。

**推荐类目**（选一个最接近的）：
- `工具` → `办公` → `待办提醒`
- `生活服务` → `综合` → `生活提醒`
- `工具` → `效率` → `日程管理`

**操作路径**：
`设置` → `基本设置` → `服务类目` → `详情` → `添加类目`

---

## 三、模板关键词（直接选用现成模板）

在订阅消息页面，搜索以下关键词，选用现成模板：

### 方案A：选用现成模板（推荐，审核快）

在模板库搜索：`待办` `提醒` `任务`

**推荐选用模板**（任选一个，关键词匹配即可）：

| 模板名称 | 关键词组合 | 适用场景 |
|---------|-----------|---------|
| 待办提醒通知 | 待办标题、提醒时间、备注 | ✅ 最推荐 |
| 任务提醒通知 | 任务名称、提醒时间、备注说明 | ✅ 推荐 |
| 日程提醒 | 日程内容、提醒时间、地点 | ✅ 可用 |

**选用步骤**：
1. 在模板库搜索上述关键词
2. 找到合适的模板，点击 **"选用"**
3. 勾选需要的关键词（至少选2个，最多选5个）
4. 提交后**立即获得模板ID**（无需审核）

---

### 方案B：申请自定义模板（如现成模板不满足需求）

**申请话术模板**（在"申请模板"页面填写）：

```
【模板标题】
待办事项提醒通知

【模板内容】
{{phrase1.DATA}}
您有一条待办事项提醒：
{{thing1.DATA}}
{{time1.DATA}}
{{thing2.DATA}}

【关键词】
待办状态 / 待办标题 / 提醒时间 / 备注说明
```

**申请理由**（填写在申请表单中）：

```
本小程序为个人时间管理工具，核心功能为待办事项管理。
用户设置提醒时间后，需要在指定时间通过订阅消息提醒用户处理未完成的待办事项。
模板用于发送待办提醒通知，包含待办标题、提醒时间、备注等信息。
符合《微信小程序订阅消息规范》中"待办提醒"类场景。
```

## 四、关键词选择建议

选用模板时，勾选以下关键词（按优先级）：

| 优先级 | 关键词 | 说明 | 示例值 |
|-------|-------|------|--------|
| ⭐⭐⭐ | 待办标题 / 任务名称 | 必选 | `完成项目报告` |
| ⭐⭐⭐ | 提醒时间 | 必选 | `2024-01-15 09:00` |
| ⭐⭐ | 备注说明 / 备注 | 推荐 | `截止日期前提醒` |
| ⭐ | 待办状态 | 可选 | `未完成` |

---

## 五、获取模板ID

选用/申请成功后：

1. 在 **订阅消息** → **我的模板** 中查看
2. 复制模板ID（格式类似：`abcDEF123ghIJK456lmNOP789qrSTU012`）
3. 将模板ID填入代码：

**文件**：`utils/subscribeMessage.js`

**修改位置**：
```javascript
const TEMPLATE_IDS = [
  '在这里填入您的模板ID',  // ← 替换这里
]
```

## 六、完整申请流程（图文步骤）

### Step 1：登录后台
访问 https://mp.weixin.qq.com/ → 登录小程序账号

### Step 2：进入订阅消息
`功能` → `订阅消息` → `申请开通`（如未开通）

### Step 3：选用模板
`公共模板库` → 搜索 `待办` 或 `提醒` → 选择合适的模板 → `选用`

### Step 4：配置关键词
勾选需要的关键词（建议全选）→ `提交`

### Step 5：获取模板ID
`我的模板` → 复制模板ID → 填入代码

---

## 七、审核时间

| 模板类型 | 审核时间 | 说明 |
|---------|---------|------|
| 选用现成模板 | **即时通过** | ✅ 推荐 |
| 申请自定义模板 | 1-3个工作日 | 需要微信审核 |

---

## 八、填入代码

获取到模板ID后，修改 `utils/subscribeMessage.js`：

```javascript
// utils/subscribeMessage.js 第3行
const TEMPLATE_IDS = [
  'abcDEF123ghIJK456lmNOP789qrSTU012',  // ← 替换为您自己的模板ID
]
```

**注意**：
- 一次性订阅：用户每次需要主动授权
- 长期订阅：仅限部分类目（政务、医疗、交通等），个人待办提醒通常用一次性订阅
- 建议引导用户在"提醒设置"页面主动点击授权

---

## 九、测试订阅消息

模板ID配置完成后，测试流程：

1. 打开小程序，进入 **"我的"** 页面
2. 点击 **"提醒设置"**
3. 开启提醒开关
4. 点击 **"去授权"**（会弹出微信订阅授权弹窗）
5. 允许授权
6. 在微信服务通知中查看是否收到测试消息（需要云函数 scheduler 触发）

## 十、常见问题

### Q1：找不到合适的模板怎么办？
A：选用关键词最接近的模板，关键词名称不影响实际发送内容，只要在发送时传入正确的 value 即可。

### Q2：模板审核被拒怎么办？
A：检查申请理由是否清晰，类目是否匹配。可以重新选用现成模板（无需审核）。

### Q3：可以用"长期订阅"吗？
A：个人待办提醒类小程序通常只能使用"一次性订阅"，用户每次需要主动授权。可以在提醒设置页面引导用户多次授权（微信允许一次授权多个消息）。

### Q4：模板ID填错了会怎样？
A：调用 `wx.requestSubscribeMessage` 时会报错 `200012`，提示模板ID无效。
